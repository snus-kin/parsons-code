# parsons-code

Some Python stuff to do stuff with [Parsons code](https://en.wikipedia.org/wiki/Parsons_code)

Using monophonic midi file:
```bash
~/programming/uni/math2340 $ ./contour.py $(./miditoparsons.py test2.mid)
*uuuuuuuuddddddd
                *
```
